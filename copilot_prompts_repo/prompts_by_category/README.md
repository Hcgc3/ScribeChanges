# Prompts por Categoria de Funcionalidade

Esta pasta contém prompts organizados por tipo de funcionalidade, facilitando a localização do prompt adequado para cada situação.

## Arquivos nesta pasta:

- `development.md`: Prompts para desenvolvimento de novas funcionalidades
- `maintenance.md`: Prompts para manutenção e refatoração de código
- `debugging.md`: Prompts para resolução de problemas e debugging
- `testing.md`: Prompts para geração de testes
- `documentation.md`: Prompts para documentação e explicação de código

## Como usar:

1. Identifique o tipo de tarefa que você precisa realizar
2. Abra o arquivo correspondente
3. Encontre o cenário que mais se aproxima da sua situação
4. Adapte o prompt para o seu contexto específico
5. Use o prompt no seu código ou no GitHub Copilot Chat
